Core Modules

六大模块，覆盖日常开发运维完整闭环

Terminal

GPU 加速渲染，多标签分屏，Blocks 输出分组，命令历史搜索，危险命令二次确认。

wgpu alacritty < 5ms

SSH / SFTP

连接管理器，可视化文件传输，端口转发，跳板机，批量命令执行，主机画像。

ssh2-rs SFTP ProxyJump

Database

SQL 编辑器，百万行数据网格，表结构浏览，NL2SQL 自然语言查询，数据同步与导出。

MySQL PostgreSQL Redis

Docker

容器生命周期管理，实时日志流，一键进入容器 Shell，Compose 编排与服务拓扑。

bollard Compose 日志流

Server

实时系统监控，远程文件管理，进程管理，服务管理，宝塔/1Panel 面板集成。

systemd crontab 面板API

AI Native

AI 不是聊天窗口，
是你的随行工程专家

OmniPanel 的 AI 能读取当前终端输出、数据库结构、容器状态和日志片段，给出可执行的建议 —— 不只是一段文本。

01

上下文感知

自动读取当前终端、SSH 主机、数据库连接、容器状态和日志片段

02

操作链编排

AI 拆解任务为多步操作链，用户确认后逐步执行，全程可审计

03

安全边界

AI 只建议不越权，危险操作需二次确认，生产环境保护内置设计

Use Cases

为真正的工程场景设计

新项目部署

连接服务器、上传配置、启动容器、检查日志 —— 从 SSH 到 SFTP 到 Docker Compose 到日志面板在同一工作区完成。

线上故障排查

AI 汇总当前主机、容器、日志和错误输出，生成排查路径。从监控告警到修复完成，全程不切换窗口。

多服务器维护

选中服务器分组后统一执行命令，结果按机器聚合和对比。批量文件分发、命令模板、执行结果对比。
